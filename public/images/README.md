# Panduan Upload Foto Coffee Shop

## Lokasi Foto
Simpan foto-foto coffee shop di folder: `public/images/`

## Nama File yang Dibutuhkan:
1. **shop-exterior.jpg** - Foto tampak depan coffee shop
2. **shop-interior.jpg** - Foto interior/suasana dalam
3. **shop-location.jpg** - Foto petunjuk lokasi/landmark

## Cara Upload:
1. Siapkan 3 foto coffee shop
2. Rename sesuai nama di atas
3. Copy ke folder `public/images/`
4. Jalankan `npm run build`
5. Push ke GitHub

## Ukuran Rekomendasi:
- Resolusi: 1200x900px (ratio 4:3)
- Format: JPG atau PNG
- Ukuran file: < 500KB per foto
- Kualitas: 80-90%

## Contoh Foto yang Bagus:
- **Exterior**: Tampak depan dengan signage jelas
- **Interior**: Suasana cozy dengan pencahayaan bagus
- **Location**: Foto jalan/landmark untuk petunjuk arah

## Setelah Upload:
Foto akan muncul di halaman home setelah hero section dengan caption:
- Tampak Depan Coffee Shop
- Suasana Interior
- Petunjuk Lokasi
